# UX Structure

Primary Tabs:
1. Players
2. Setup
3. Teams

Secondary Page:
- Library (saved presets, results, export/import)

## Players Tab
- Select player-set
- Add/edit/delete player
- Edit criteria
- Manage player-sets

## Setup Tab
- Select or create preset
- Set number of teams
- Choose and rank criteria
- Generate button

## Teams Tab
- Generate / Re-roll
- Display teams
- Fairness score
- Manual override
- Result history

## Library
- Saved presets
- Saved results
- Export / Import
