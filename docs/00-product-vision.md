FairDraft is an offline-first Progressive Web App (PWA) that generates fair teams from a list of players using soft, ranked criteria.

## Core Promise
- Works offline after install
- No account required
- No backend database
- All data stored locally on device
- Fast and simple for real-world usage (field, gym, classroom)

## Target Users
- Rec league captains
- Coaches
- PE teachers
- Intramural organizers
- Casual group organizers

## Key Principles
- Simplicity first
- Soft constraints (never fail generation)
- Transparent fairness score
- Manual override allowed
- Portable via export/import JSON
